Reverse the array

Explanation:
Function reverseArray(arr []int) []int:
This function takes an array of integers as input and returns the reversed array.
Two pointers, left and right, are initialized to point to the beginning and end of the array, respectively.
A for loop runs until the left pointer is less than the right pointer. Inside the loop, the elements at the left and right positions are swapped, and the pointers are moved towards the center.
Main Function:
An example array is defined.
The original array is printed.
The reverseArray function is called to reverse the array.
The reversed array is printed.
